# FGC-REQ-SPEC-001 - SRS Document Registry
Version: 1.0

## Description
Provide a simple on-disk SRS registry that discovers definition files in `docs/srs/` and enables lookup by requirement ID.

## Rationale
Centralized lookup ensures tools can validate requirement references consistently.
