# Glossary & Synonyms (repo‑local)

-   **SCMP** ↔ **CM Plan**: Software Configuration Management Plan (IEEE 828 / ISO 10007 content).
-   **Baseline** ↔ **Configuration baseline**: Tagged state approved under CM (e.g., `vX.Y.Z`).
-   **FCA/PCA**: Functional/Physical Configuration Audit (evidence & build/package checks).
-   **RTM**: Requirements Traceability Matrix/records (Req ↔ Test ↔ Code).
-   **ADR**: Architecture Decision Record.
-   **Verification**: Did we build it right? (meets specified requirements).
-   **Validation**: Did we build the right thing? (meets stakeholder needs).
-   **Information item** (15289): A required document or record (e.g., SRS, Test Report, Release Record).
