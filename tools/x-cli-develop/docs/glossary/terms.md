# Glossary (Concise)

-   SCMP ≈ CM Plan (Configuration Management Plan).
-   Baseline ≈ Configuration baseline (tagged SemVer + evidence set).
-   RTM ≈ Requirements Traceability Matrix.
-   FCA/PCA ≈ Functional/Physical Configuration Audit.
-   Change Control ≈ Review/approval workflow tied to required status checks.
-   Verification vs Validation ≈ “Build the product right” vs “Build the right product”.
