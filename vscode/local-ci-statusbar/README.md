Local CI Status Bar
===================

Adds three status bar buttons to quickly run Local CI dev-mode tasks:

- Enable Dev Mode (Stage 25)
- Disable Dev Mode (Stage 25)
- Toggle Dev Mode (Stage 25)

Usage (Extension Development Host)
- Open this workspace in VS Code.
- Run the debug configuration "Run Extension: Local CI Status Bar".
- In the Extension Development Host window, use the status bar buttons on the left.

Alternatively, package/install as a VSIX if desired, then the buttons will appear in your main window.
