# Summarize-PRVIHistory.ps1

**Path:** `tools/Summarize-PRVIHistory.ps1`

## Synopsis
Converts `pr-vi-history-summary@v1` JSON into a Markdown table and optional enriched JSON for PR comments or workflow step summaries.

## Description
- Reads the summary generated by `Invoke-PRVIHistory.ps1`, builds aggregate totals (targets, completed, comparisons, diffs), and renders a Markdown table showing each VIâ€™s change type, comparison counts, diff count, status, and report links.
- When `-MarkdownPath` is provided, writes the Markdown to disk; `-OutputJsonPath` saves an enriched object (`totals`, `targets`, `markdown`).
- Appends key fields to `$GITHUB_OUTPUT` (`markdown`, `markdown_path`, `target_count`, `diff_count`, etc.) so GitHub Actions can surface the summary.

### Parameters
| Name | Type | Default | Notes |
| --- | --- | --- | --- |
| `SummaryPath` | string (required) | - | Path to `vi-history-summary.json`. |
| `MarkdownPath` | string | - | Destination for the rendered Markdown. |
| `OutputJsonPath` | string | - | Optional enriched JSON output. |

## Outputs
- Markdown table, optional enriched JSON, and GitHub output variables.
